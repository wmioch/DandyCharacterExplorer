================================================================================

1. Go to your repository on GitHub
2. Click Settings (top right)
3. Left sidebar → Pages
4. Source: Deploy from a branch
5. Branch: main
6. Folder: / (root)
7. Click Save
8. WAIT 1-2 minutes
9. GitHub will show: "Your site is live at https://YOUR-USERNAME.github.io/..."

STEP 5: SUBMIT TO GOOGLE (10 minutes)
================================================================================

A) Create Google Search Console property:
   1. Go to google.com/search-console
   2. Click "Add property"
   3. Enter: https://YOUR-USERNAME.github.io/DandyCharacterExplorer/
   4. Click Continue
   5. Verify ownership (choose HTML tag or GitHub connection)
   6. Click Verify

B) Submit sitemap:
   1. In Search Console, click "Sitemaps" (left sidebar)
   2. Enter: https://YOUR-USERNAME.github.io/DandyCharacterExplorer/sitemap.xml
   3. Click Submit
   4. Check Coverage report (may take 24-48 hours to process)

================================================================================
TROUBLESHOOTING
================================================================================

"My site doesn't appear in Google yet"
  → Normal! Takes 2-4 weeks. Check Google Search Console for errors.

"Sitemap shows errors"
  → Verify YOUR-USERNAME is correct in all URLs
  → Check file is in root directory (not in a folder)

"Social media preview isn't showing"
  → Clear cache: facebook.com/tools/debug/og/object/
  → May take 24 hours to refresh

"robots.txt returns 404"
  → Verify it's in the root directory
  → Wait 5 minutes for GitHub Pages to rebuild
